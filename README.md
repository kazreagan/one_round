## OneRound Pictures — Website Template

A Cinematic web experience crafted for OneRound Pictures, featuring a responsive hero slider, directors showcase, films grid, a custom video modal, and a mobile friendly navigation system.
Built with simplicity in mind — pure `HTML`, `CSS` and `JavaScript`.




## Features

1. Hero Slider

- Auto/manual sliding

- Cinematic overlay and typography

- Arrow controls

- Each slide highlights a film title and director


2. Responsive Navigation

- Clean desktop menu

- Mobile hamburger toggler

- Slide-in menu animation

- Auto-close behavior when selecting a link


3. Film Grid

- 6 cards showcasing film thumbnails

- Local MP4 video support

- Smooth hover effects

- Click-to-open modal video player


4. Video Modal

Supports:

- Local .mp4 files

- YouTube/Vimeo embeds

- Click-outside or X-button to close

- Auto-stop playback on close


5. Directors Section

- Stylish image bar layout

- Quick link to view each director’s films

- Responsive row display


6. Footer Section

- Quick links

- Contact info

- Social media icons

- Automatic current-year rendering


7. Mobile Menu Toggler

The site includes a custom built mobile menu toggler with a sliding navigation panel for screens under 880px.

Implemented with:

- A 3-bar animated “hamburger”

- Smooth CSS transform transitions

- JS click toggle + link auto-close


## Tech Stack

- HTML5

- CSS3

- Google Fonts (Montserrat, Playfair Display)

- FontAwesome Icons


## Vanilla JavaScript

No frameworks — lightweight and fast


## Setup & Usage

1. Clone the repository:
```bash
git clone https://github.com/yourusername/one_round.git
```


2. Open the project folder:
```bash
cd one_round
```


3. Launch index.html in your browser.

No build steps. No bundlers.
Just run and enjoy.

## Customization

You can easily adjust:

- Slider timings

- Menu breakpoint

- Overlay colors

- Card grid count

- Video sources and thumbnails

- Director list

- Services and page sections


`Just edit the HTML and the assets folder.`
